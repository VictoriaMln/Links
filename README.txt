Сервис для генерации коротких ссылок.

- Консольное приложение.
- По умолчанию создаётся короткая ссылка вида shrt.com/..., срок активности ссылки - 24 часа.
- Количество доступных переходов по ссылке устанавливается пользователем.
- Каждый пользователь имеет свой уникальный ID (UUID), короткие ссылки привязаны к нему.
- Данные хранятся в оперативной памяти только во время сессии, выгрузка в файл или в БД не реализована.

Класс User:
- имеет поле userId - уникальный идентификатор пользователя (UUID) и список ссылок, принадлежащих пользователю - links.

Класс ShortLink:
- описывает короткую ссылку, содержит данные по исходной ссылке, максимальному количеству переходов, счётчику переходов, времени создания и времени окончания действия ссылки;
- методы для проверки активности ссылки.

Класс LinkManager:
- отвечает за хранение всех пользователей и их ссылок (в Map<String, User>);
- создание или поиск пользователя;
- создание короткой ссылки (createShortLink) - тут можно изменить время окончания действия ссылки и её вид ;
- открытие короткой ссылки в браузере;
- удаление просроченных ссылок или ссылок, у которых вышел лимит по количеству переходов;
- удаление ссылок по запросу пользователя;
- редактирование количества переходов по ссылке.

Класс Main:
- содержит точку входа;
- консольное меню, предлагающее создать, открыть, удалить короткую ссылку, показать все ссылки, принадлежащие данному пользователю, отредактировать ссылку или выйти;
- при выборе в меню пунктов 1 (создать), 2 (открыть), 3 (показать все), 4 (удалить) или 5 (редактировать) сначала проводится удаление неактивных ссылок, затем запрашивается UUID. Дальнейшие действия производятся только с активными ссылками, принадлежащими данному UUID.
